Part 3:Configuration Management Tools and Recipes

There is an overwhelming list of tools in the Kubernetes configuration management ecosystem where we don't have clear guidance on the trade-offs. This part of the book will guide cloud-native practitioners in choosing the tools for Kubernetes configuration automation that best suit the use case.

This part comprises the following chapters:

  • Chapter 8, Knowing the Trade-Offs
  • Chapter 9, Using Helm, Kustomize, and KubeVela
  • Chapter 10, Onboarding Applications with Crossplane
  • Chapter 11, Driving the Platform Adoption
